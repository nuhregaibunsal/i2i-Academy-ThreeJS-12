# i2i-Academy-ThreeJS-12

A basic 3D obstacle avoider game built with **React Three Fiber** and **@react-three/drei**, developed as part of the i2i Academy internship assignment.

## Tech Stack

- React 18 + Vite
- three.js
- @react-three/fiber
- @react-three/drei

## Getting Started

```bash
npm install
npm run dev
```

Then open the printed local URL (default: http://localhost:5173).

## Controls

- **A / D** or **← / →** — move the player horizontally
- Avoid the incoming red blocks
- Each block you dodge increases your score
